## 🚀 Featured Projects

### 🥛 [MilkMan (KP Fresh Dairy)](https://github.com/kaustubh023/MilkMan)
A full-stack dairy delivery & subscription management platform with role-based access control.

- **Stack:** Django REST Framework · React 19 · Vite · Tailwind CSS · JWT Auth · SQLite
- **Features:** Multi-role system (Customer / Admin / Staff), cart & checkout, subscription management, admin dashboard
- **Architecture:** Decoupled backend API + React SPA with JWT-based auth and role-aware routing

---

### 🏦 [Loan Approval Prediction System](https://github.com/kaustubh023/loan_approval_Project)
An ML-powered loan eligibility predictor with an interactive Streamlit web app.

- **Stack:** Python · Scikit-learn · Streamlit · Pandas · Plotly
- **Accuracy:** 89.68% on 45,000 records from Kaggle
- **Features:** Interactive gauge chart, key factor analysis, dark-themed UI, confusion matrix & feature importance visualizations
